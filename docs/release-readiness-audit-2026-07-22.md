# Аудит готовности к реализации — 22.07.2026

## Итог

Проект приведён в состояние готовности к реализации: критические функциональные и PWA/Android-проблемы исправлены, мобильный интерфейс проверен, unit-тест и Android lint пройдены, debug и release-варианты собраны.

## Исправления

- Устранён конфликт `id="count"`: счётчик повторов и переключатель голосового отсчёта теперь независимы.
- Переключатель голосового отсчёта получил корректные `role="switch"`, `aria-checked` и управление с клавиатуры.
- Скрытые полноэкранные панели исключаются из фокуса и дерева взаимодействия через `inert`/`aria-hidden`.
- Возвращено масштабирование страницы; минимальная высота интерактивных элементов повышена до 44 px.
- Service worker кэширует только успешные same-origin ответы и возвращает HTML-fallback только для навигации.
- Версия PWA-кэша повышена до `pushup-v21`.
- В Android отключены mixed content, cleartext traffic и резервное копирование пользовательских данных.
- Добавлены правила Android 12+ для запрета cloud backup и device transfer.
- Удалено избыточное разрешение `USE_EXACT_ALARM`; оставлено запрашиваемое пользователем `SCHEDULE_EXACT_ALARM`.
- В release-сборке включены R8 и resource shrinking.
- Устаревшие вызовы оформления системных панелей заменены на edge-to-edge API.
- Исправлены adaptive/monochrome/legacy/round иконки приложения.
- Шаблонный бело-голубой splash Capacitor заменён фирменным графитово-лаймовым во всех плотностях и ориентациях.
- Удалены действительно неиспользуемые шаблонные Android-ресурсы; динамически используемые ресурсы защищены от shrinker.

## Единственное внешнее замечание

Android lint сообщает, что доступен Gradle 8.14.5 вместо 8.14.3. Обновление было проверено, но загрузка дистрибутива дважды оборвалась внешним 10-секундным сетевым таймаутом wrapper. Проект оставлен на полностью проверенной 8.14.3; это не ошибка приложения и не блокирует сборку. Для публикации остаётся только подписать release APK/AAB секретами CI или локальным release-keystore.
